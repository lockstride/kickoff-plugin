# Reference Templates

These Pandoc reference templates control the styling of generated external documents.

## Files

- **reference.docx** - Template for all document types (Business Brief, Market Analysis, etc.)
- **reference.pptx** - Template for pitch deck presentations

## How Templates Work

When generating external documents, these templates are copied to the startup's `external/` folder if not already present. Users can customize the copied templates to adjust styling for their specific startup.

Pandoc uses these reference documents to extract styles:
- Heading styles (Heading 1, 2, 3, etc.) — left-justified
- Body text formatting
- List styles
- Table formatting
- Page/slide layout
- Footer content

## Default Styling

The baseline templates use:
- **Left-justified** headings and titles (not centered)
- Clean, professional typography
- Minimal spacing around titles
- Header displays doc version
- Footer: "Generated by Lockstride Kickoff" (10pt, light gray, italic)

## Metadata Fields

Metadata is embedded in YAML front matter in the intermediate markdown file:

```yaml
---
title: PayFlow
subtitle: Business Brief
version: v2601261921
subject: Business Brief
keywords: PayFlow, Business Brief
---
```

These fields are available in the template:
- `title`: Startup name — rendered via Title style in body
- `subtitle`: Document type — rendered via Subtitle style in body
- `version`: Version string — set as custom document property
- `subject`: Document type (for document properties)
- `keywords`: Startup name and document type (for document properties)

## Version Placeholder

The template header contains `{{VERSION}}` as a placeholder text. After Pandoc generates the document, post-processing replaces this placeholder with the actual version string (e.g., `v2601261921`).

If you customize the template and want the version in the header:
- Keep the text `{{VERSION}}` where you want the version to appear
- The post-processing step will replace it automatically

## Customizing

To modify the templates in the startup's `external/` folder:

1. Open the file in Word (for .docx) or PowerPoint (for .pptx)
2. Modify styles via the Styles pane (Heading 1, Heading 2, Body Text, etc.)
3. Adjust footer text if needed
4. Save the file — future documents will use your styles

See: https://pandoc.org/MANUAL.html#option--reference-doc
